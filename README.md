## 🔐 Projeto Login

Este projeto consiste em uma página de login moderna e funcional, desenvolvida com foco em simplicidade, usabilidade e design limpo. A aplicação simula um sistema de autenticação onde o usuário pode acessar sua conta por meio de credenciais como e-mail e senha.

A interface apresenta campos essenciais para login, além de opções auxiliares como recuperação de senha e criação de conta, proporcionando uma experiência completa ao usuário. Esse tipo de página é fundamental em aplicações web, pois é responsável por validar o acesso e proteger informações privadas, garantindo que apenas usuários autorizados consigam entrar no sistema .

## 🚀 Funcionalidades

- Campo para e-mail e senha
- Botão de acesso (login)
- Opção de recuperação de senha
- Link para criação de conta
- Layout responsivo e intuitivo

## 🎯 Objetivo

O principal objetivo do projeto é praticar e demonstrar habilidades em HTML, CSS e JavaScript, criando uma interface realista de autenticação utilizada em diversos sistemas web.

## 💡 Importância

Páginas de login são essenciais em qualquer plataforma digital, pois funcionam como uma camada de segurança, impedindo acessos não autorizados e protegendo dados dos usuários.

## 🔮 Tecnologias Utilizadas

- HTML 🟧
- CSS 🟦
